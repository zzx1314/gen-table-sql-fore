# gen-table-sql

这是一个使用vue3生成mysql表sql的vue项目

## 项目启动

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
